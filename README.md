# Zuari Employee Travel Desk (Frontend)

Login/Signup screen for an internal employee travel-request portal:
- Employees raise travel requests and enter required details.
- Managers approve requests.
- Tickets for train/flight are raised post-approval.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
